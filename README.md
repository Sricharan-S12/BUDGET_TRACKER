# BUDGET_TRACKER
This Budget Tracker is a multi-profile expense management application built with Tkinter and TTKBootstrap .A multi-profile Budget Tracker for students to monitor spending, set limits, track savings, add expenses, and view graphs. Helps students control impulsive spending and understand their expense patterns easily.
